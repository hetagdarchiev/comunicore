# 📝 Feature: Text Editor (Markdown)

Мощный и гибкий Markdown-редактор, построенный по методологии **FSD**, с поддержкой Drag-and-Drop, вставки из буфера обмена и безопасным рендерингом.

## Основные возможности

### Инструментарий (Toolbar & Shortcuts)
Реализован полный набор инструментов для форматирования. Логика вынесена в сервисный класс `MdEditor`, что обеспечивает легкую расширяемость.
- **Заголовки**: Умное управление уровнями (H1-H6).
- **Акценты**: Жирный (**Bold**), Курсив (*Italic*), Зачеркнутый (~~Strike~~).
- **Блоки**: Цитаты (`> Quote`), Код (inline и блоки ```).
- **Списки**: Упорядоченные (`1. `) и маркированные (`- `).
- **Ссылки**: Интеллектуальная обработка URL.

### Работа с медиа (Media Handling)
Бесшовная интеграция загрузки изображений:
- **Drag-and-Drop**: Загрузка файлов при перетаскивании.
- **Clipboard Paste**: Поддержка `Ctrl+V` для скриншотов из буфера.
- **Smart Insert**: Генерация Markdown-шаблона `![alt](url)` и вставка точно в позицию курсора с учетом контекста строк.

### Управление и UX
- **Hotkeys**: Обработка через `MdEditorKeyBoards` (поддержка `Ctrl+B`, `Ctrl+I`, `Ctrl+K` и др.).
- **Draft System**: Автосохранение черновика в `localStorage` (защита от потери данных).
- **Preview Mode**: Мгновенное переключение между редактором и отрендеренным HTML.

## Безопасность
Для предотвращения XSS-атак используется многоуровневая обработка:
1. **markdown-it**: Парсинг Markdown в HTML.
2. **DOMPurify**: Глубокая очистка (sanitization) перед рендерингом.
3. **dangerouslySetInnerHTML**: Безопасный вывод проверенного контента.
